Author : Heinz Sait


Objective:

  • The objective of the game is to make a shooting game.
  • The game also defines bullet management.


Logic Design and Implementation:

    • The aliens, bunkers, ufo and Player Ship are created using bit-mapping.
    • The no. of bullets are fixed. They are used in a loop to create an illusion of infinite bullets.
    • The player has 3 lives.
