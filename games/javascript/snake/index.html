Author : Heinz Sait


Objective:

  • The objective of the game is to make a multiplayer game.
  • There are two snakes controlled by their controls : -w,a,s,d & up,left,down,right.
  • The last man standing wins.


Logic Design and Implementation:

    • There are for types of power-up.
    • Blue - Increase Speed
    • Yellow - Decrease Speed
    • Green - Increase Size
    • Red - Decrease Size
